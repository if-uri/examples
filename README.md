# ifURI examples

Runnable examples for `urirun` and ifURI-style URI workflows.

This repository was split out from:

```text
tellmesh/urihandler/v2/examples
```

## Examples

- `artifacts/` - scanning Dockerfile, Makefile, package metadata and manifests
- `decorators/` - Python decorator-generated URI bindings
- `device_mesh_lab/` - dashboard with device agents, URI flow execution and `browser://` demo routes
- `docker_uri_flow/` - Docker Compose services communicating through URI resources
- `generators/` - JS, Node.js, TypeScript and PHP binding generation examples
- `html_uri_app/` - browser UI calling a Python backend through URI actions
- `json/` - binding document examples
- `multi_transport/` - HTTP and gRPC worker transport demo
- `novnc_lan_flow/` - multi-computer noVNC LAN workflow demo
- `python/` - Python adapter tests and examples
- `transports/` - local, queue, serverless, HTTP and gRPC transport examples
